任務目標：打造類似 Hello Kitty 的日系超可愛療癒風介面，並將首頁依據過期狀態分成四大區塊。

請編寫 `index.html` 與 js/ 目錄下的 `ui.js`：
1. index.html 視覺規範：
   - 網頁底色改為奶油白 (#FFFDF9)。
   - Header 改為溫柔粉紅色 (bg-pink-400)，標題為「🍏 冰箱保衛戰 🎀」。
   - 主畫面上方保留一個 App 內警告橫幅（例如：「今天有 X 項食材快過期囉！」）。
   - 主畫面依序呈現四個大圓角卡片區塊（區塊邊框：border-4 border-pink-200，rounded-3xl）：「🚨 救命呀 (已過期/快過期)」、「🐱 快吃我」、「🎀 安全唷」、「✔ 已完食」。
   - 右下角設計一個粉紅色圓形懸浮 `＋` 按鈕。

2. js/ui.js 渲染邏輯：
   - 實作 `renderApp(foods)`：調用 `engine.js` 的分類函數，將食材分別渲染到上述四個區塊。
   - 卡片外觀：顯示食材名稱、數量 (e.g., x2)、存放位置圖示（❄冷凍庫、🥛冷藏、🏠常溫），以及剩餘天數文字（若過期則顯示「已過期 X 天」）。
   - 每個未完食卡片右側有一個大大的可愛「吃完了」按鈕。
   - 實作新增食物彈窗 (Modal)，點選食材類別（肉類 🥩、奶蛋 🥛 等）時，會動態連動日期選擇器。

完成後請整合代碼並儲存。
